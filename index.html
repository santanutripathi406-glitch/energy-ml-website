About the Project

This research focuses on predicting transformer failures using a lightweight machine learning model designed for rural power distribution systems. The goal is to improve reliability while minimizing computational cost.

Problem Statement

Rural transformers often fail due to overload, overheating, and irregular maintenance. Early prediction can reduce power outages and maintenance costs.

Abstract

Transformer failures in rural power distribution networks often lead to prolonged outages and increased maintenance costs. This project presents a lightweight, energy-efficient machine learning model designed to predict transformer failure risk using minimal computational resources. The system utilizes operational parameters including temperature, load percentage, and voltage fluctuation to classify transformer health into risk categories. The approach prioritizes deployability in low-resource environments while maintaining reliable predictive performance.

Methodology

  • Collection of transformer operational parameters
  • Feature selection for efficient prediction
  • Lightweight ML-based risk classification
  • Visualization of predicted failure risk

Dataset Description

The dataset represents operational behavior of distribution transformers under normal and stressed electrical conditions. The structure is based on standard monitoring parameters used in power distribution systems.

Input Features

  • Transformer temperature (°C)
  • Load percentage (%)
  • Voltage fluctuation (%)
  • Operating condition indicators

Data Characteristics

  • Time-series sensor-based measurements
  • Includes safe, warning, and critical operating ranges
  • Failure risk derived using engineering safety thresholds

Preprocessing

  • Data normalization
  • Threshold-based labeling
  • Feature selection for lightweight computation

The dataset structure reflects practical transformer operating conditions used in predictive maintenance studies and supports deployment in low-resource rural environments.

Model Architecture

The proposed system uses a lightweight classification approach optimized for low computational overhead and real-time prediction capability.

  • Low memory and energy consumption
  • Fast inference suitable for edge deployment
  • Interpretable risk classification

Prediction Output

  • Low Risk — Normal condition
  • Medium Risk — Warning condition
  • High Risk — Potential failure

Results and Accuracy

The model demonstrates reliable prediction performance while maintaining low computational requirements. The system effectively identifies transformer risk conditions using minimal input parameters, making it suitable for rural deployment environments.

  • High prediction reliability
  • Low computational cost
  • Fast real-time prediction capability

Future Work
